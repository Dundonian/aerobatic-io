Features

Smart hosting for web apps

Highly optimized asset delivery

Aerobatic optimizes asset delivery for lightning fast downloads. Plus we'll also automatically perform pre-processing of CoffeeScript, Sass, LESS, Stylus, Jade, Markdown, and more.

Use the libraries you love

Aerobatic is a platform, not a framework. You can build your app with any browser technology you wish including jQuery, Bootstrap, AngularJS, Ember, D3, and whatever next best thing comes along.

Split test like a pro

Not sure how users will react to your app’s latest version? Split your traffic with a simple slider and watch the results in real-time.

Dev friendly tools

The yoke command line tool cleanly integrates into a streamlines modern web dev workflow. Push a new version to production with just a few keystrokes.

Works with the web

Rich interactivity of a single page app with the discoverability and reach of the web. Multi-device targeting, SEO, permalinks, and more.

Built-in Security

Authenticate users with OAuth or against your own credential store. Enforce SSL with the click of a box.

Integrate with any backend

Invoke web APIs directly from the browser or use our intelligent service proxy to securely store and manage your access keys.

Goodbye test environment

With Aerobatic’s simulator mode, run your front-end changes locally, and the rest of your app in the cloud. Preview your changes live in production.
